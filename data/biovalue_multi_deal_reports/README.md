# BioValue Multi-Deal Reports

This archive contains worked-example rNPV analyses for 11 pharmaceutical acquisitions (2023-2025).

## Contents

Each deal includes:
- **Excel workbook** (`*_worked_example.xlsx`) with 3 tabs: Deal Summary, rNPV Model, Valuation Gap Analysis
- **Markdown summary** (`*_analysis_summary.md`) with deal overview, valuation results, and interpretation

## Methodology

All valuations use risk-adjusted NPV (rNPV) methodology with:
- Bear/Base/Bull scenario modeling
- Phase-appropriate probability of success (PoS)
- Therapeutic area-specific discount rates (WACC)
- Peak sales projections based on indication epidemiology and market comps
